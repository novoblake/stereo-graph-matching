# stereo-graph-matching
stereo pipeline that operates on superpixels. Graph Attention Networks. Three variants  - `stereodepth1.py` – baseline using Fibonacci and GraphSAGE. - `stereog.py` – enhanced architecture with a ResNet‑50, ASPP, cost MLP, GAT refinement, and U‑Net. - `strg.py` – experimental version adding unsupervised losses.
